# Credit Card User Segmentation  

---

## **Descriptions**  

This project segments credit card users into different personas using clustering and ensemble techniques. <br>

---

## **Technologies**  

- Python <br>
- Pandas <br>
- Scikit-learn <br>
- Matplotlib <br>

---

## **Methodology**  

- Applied KMeans and Agglomerative clustering <br>
- Combined results using ensemble method <br>
- Created personas: <br>
  - High Spender <br>
  - Risky User <br>
  - Low Activity <br>
  - Regular User <br>

---

## **Output**  

- Scatter plot of customer segmentation <br>
- Persona distribution graph <br>
- Final dataset with clusters and personas <br>

---

## **How to Run**  

pip install -r requirements.txt <br>
python src/main.py
